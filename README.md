# Description

## Dev

1. Clonar repositorio
2. Copiar el archivo ````env.template````  y renombrar a ````.env````
3. Ejecutar
   ```
   npm install
   ```
4. Levantar la imagen de docker ejecutando el comando
   ```
   docker-compose up -d
   ```
5. Iniciar servidor NestJS con el comando
   ```
    npm run start:dev
   ```
6. Visitar el sitio 
   ```
   localhost:3000/graphql
   ```
